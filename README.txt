新建遠端倉庫
然後將本地項目上傳至遠端倉庫
1、git init
2、git add .
3、git commit -m "提交信息"
4、git remote add origin <your-remote-repository-url>
5、git push origin master
-------------------------------------------------------
如果是下載遠端倉庫項目
1、git clone <repository_url>
-------------------------------------------------------
如果需要从远程仓库拉取最新的代码
1、git pull <remote> <branch>
